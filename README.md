# nginx-wordpress-cookbook
My go-to NGINX configuration files for running WordPress on NGINX.


Designed for: 
 - WordPress on NGINX
 - WP Super Cache
 - Let's Encrypt SSL
 - Diffie-Hellman
 - Uses Quad9 for DNS Resolver https://www.quad9.net/#/
 - Redirects non HTTPS traffic to HTTPS
